# React User Cards App

This project was built while learning React fundamentals.

## Features

- Create user cards using a form
- Display multiple cards dynamically
- Delete cards from the list
- Form handling using React Hook Form
- State management with React Hooks (`useState`)
- Props and Prop Drilling implementation
- Dynamic rendering using `map()`
- Event handling in React

## Concepts Practiced

### React Components
- Functional Components
- Reusable Card Component
- Form Component
- Cards Container Component

### State Management
- Managing user data with `useState`
- Updating arrays in state
- Removing items from state

### Props & Prop Drilling
Data is passed from the parent component (`App`) to child components (`Cards` and `Card`) using props.

### React Hook Form
Used `react-hook-form` for:
- Form registration
- Form submission
- Form reset
- Input value handling

## Project Flow

1. User fills out the form.
2. Form data is submitted.
3. A new card is created and displayed.
4. User can delete any card using the Remove button.
5. State updates automatically and the UI re-renders.
